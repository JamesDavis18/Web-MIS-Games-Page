About: How I chose these games

For this dashboard website I wanted a good mixture of easiluy embeddable web games, that have a different variety of playstyles so that a large group of users is able to use this website and find something that they enjoy. When choosing the games for this website dashboard, I would either choose them based of of previous experience palying said flash game, use suggestions form other users that have played a flash game that they recommend and that I feel fits into the website. Other times I tried completely new games and added them if they met my criteria.

  • Must be fun to play
  • Musn't be too additive
  • Must be fully embeddable within the website
  • Must work within the website and not link you to the source website
  • Musnt be too difficult (The website/dashboard was designed with younger users in mind).

For the dashboard to be sucessful, it must use a colour palette that is accesible for users, use an-easy-to-read font and allow users to easily chose the game(s) they want from a selection.
